01 flagship

Real-time CDC: Postgres → Kafka → Iceberg

Database changes land in the lakehouse in seconds, not overnight. Debezium captures Postgres WAL events, Kafka + Schema Registry govern delivery, and Spark Structured Streaming upserts into Apache Iceberg with MERGE INTO — idempotent, compacted, and fully reproducible with one docker compose up.

Debezium Kafka Iceberg Spark Streaming Docker
-- exactly-once upsert
MERGE INTO lake.orders t
USING changes c
ON t.id = c.id
WHEN MATCHED THEN UPDATE
WHEN NOT MATCHED THEN INSERT
✓ 12k events/min · sub-minute lag
